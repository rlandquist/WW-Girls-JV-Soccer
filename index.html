How this works

All three tools save to a single GitHub repository so your data stays in sync between phone, tablet, and laptop — and survives browser cache clears. Edit on one device, the others pick up the change next time you open them.

No backend, no subscription, no ads. Just static files served from GitHub Pages talking to the GitHub REST API on your behalf.
